## 1. Introduction

PilotSeal operates [pilotseal.com](https://pilotseal.com).

PilotSeal is an aviation tools and reference website.

We do not sell user data, run advertising networks, or build behavioral profiles of users.

## 2. Account & Login Features

Some features of PilotSeal require login access.

When users create or use an account, limited account-related information may be processed for authentication and saved features.

This may include:

- Email address
- Authentication identifiers
- User-saved data related to site features

This information is only used to operate account functionality and saved user features.

## 3. Third-Party Services

PilotSeal uses third-party infrastructure providers to operate the website. Current providers may include:

- [Supabase](https://supabase.com) — authentication and backend services
- [Cloudflare](https://www.cloudflare.com) — security and network performance
- [Netlify](https://www.netlify.com) — hosting and deployment
- [Google Analytics](https://analytics.google.com) — basic traffic analytics

These services may automatically process limited technical information required for normal website operation, such as request metadata, device/browser details, and session-related information.

## 4. Analytics

PilotSeal uses Google Analytics to understand general website usage and improve the site.

PilotSeal itself does not use analytics data to identify individual users.

More information:

- [Google Privacy Policy](https://policies.google.com/privacy)

## 5. Cookies

Some parts of the website may use cookies or similar technologies for:

- Login sessions
- Security
- Basic site functionality
- Analytics

Users can control cookies through browser settings.

## 6. External Links

PilotSeal may contain links to third-party websites, FAA resources, PDFs, or external services.

We are not responsible for the content or privacy practices of external websites.

## 7. Data Security

Reasonable measures are used to protect account-related information and site infrastructure.

However, no online system can guarantee absolute security.

## 8. Changes

This Privacy Policy may be updated periodically.

Updates become effective once published on this page.
